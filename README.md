# 🐱 pet-beamer — 节奏梳毛游戏

> 一款融合 **节奏音游** 与 **虚拟宠物** 的网页小游戏  
> 用鼠标或键盘给猫咪梳毛，跟上节拍，赢取高分，解锁更多猫咪和歌曲！

## ✨ 特性

- 🐈 **多只猫咪可选**  
  内置 10 种不同性格的猫咪，每只猫的“警觉度”和“回头频率”各不相同。

- 🎵 **双模式玩法**
  - **自由模式**：匀速梳理猫咪，小心它突然回头抓伤你。
  - **节奏模式**：跟随节拍按下 `J/F`（普通拍）或 `K/D`（重拍），空拍时及时收手，避开“回头”与“偷看”。

- 🛠️ **高度可自定义**
  - 上传本地图片，制作专属猫咪（平静图 & 警惕图）。
  - 导入音频文件，自动分析 BPM 并生成节拍谱面（支持手动编辑）。
  - 更换梳子、锄头等工具，甚至上传自定义工具图片。

- 💾 **本地持久化**  
  所有自定义猫、歌曲、工具数据均存储在 IndexedDB / LocalStorage 中，关闭页面不会丢失。

- 🔊 **沉浸式音效**  
  梳毛沙沙声、节拍提示音、连击欢呼声，全部由 **Web Audio API** 实时合成。

## 🎮 游戏玩法

### 🕹 自由模式
- 按住鼠标左键（或手指触摸）在猫咪身上移动。
- 速度太慢 → 降低好感度  
- 速度太快 → 激怒猫咪
- 屏幕实时显示「舒服 / 警惕」状态条：
  - 舒适度满：获得额外分数
  - 警惕度满：被猫爪攻击
- 猫咪会随机回头，**必须立刻停止梳毛**，否则直接失败。

---

### 🎵 节奏模式
1. 点击「节奏模式」进入。
2. 选择内置歌曲，或导入音频并生成谱面。
3. 根据节拍轨道输入对应按键：

| 节拍类型 | 按键        |
| -------- | ----------- |
| 普通拍   | `J` / `F`   |
| 重拍     | `K` / `D`   |
| 回头事件 | 松开所有键  |
| 偷看事件 | 松开所有键  |

- 准确命中节拍 → 增加连击与分数  
- 空拍或按错 → 打断连击并提升警惕度  
- 完整演奏后结算评价：**SS ~ D**

## 🧩 自定义内容

### 🐱 添加自定义猫咪
1. 点击猫咪区域的「➕」按钮。
2. 上传 **平静状态图** 和 **警惕状态图**（推荐 PNG，透明背景）。
3. 填写名字与简介，即可在游戏中使用。

### 🎶 添加自定义歌曲
1. 点击歌曲区域的「➕」按钮。
2. 填写曲名与 BPM（支持自动识别）。
3. 手动编写节拍序列，例如：
1,0,2,S,1,0,L,0
4. 上传音频文件并自动播放。

### 🛠 更换 / 上传工具
- 点击底部工具栏的「➕」上传本地图片。
- 工具图标会影响鼠标样式及梳毛时的幽灵图像。
